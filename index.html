Como tudo começou...

A idéia surgiu na matéria de Programação e Algorítmos do Profº Alcimar. Em aula, nós criamos um jogo em Java onde o usuário fornecia um número para o programa e recebia uma mensagem informando se o número escolhido era maior ou menor que o número correto, onde estaria o Wally. Tudo com uma interface arcáica e apenas MessageBox padrão do Java.

Então tive a idéia de migrar o jogo pra Web, com uma interface visual mais amigável. Reaproveitei a lógica e parte do código Java utilizando-os no Javascript, que não possui uma sintaxe muito diferente do Java. Criei um layout para servir de base para o jogo e, aos poucos, fui otimizando a UX.

Ainda considero o jogo em fase alpha pois não está perfeito. Ele possui alguns problemas não muito graves e algumas coisas incompletas. Também não é responsivo e não possui compatibilidade cross-browser.
